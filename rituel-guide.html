SVG icon process — follow every time

1
Provide a reference image. Always ask for or provide a reference before starting. The closer the reference, the faster the result.
2
Draw everything upright, top to bottom. Design the full icon vertically using only simple primitives: rect, ellipse, line. Avoid freehand path for the main body. No rotation group needed.
3
Build in order: bulb → flange → barrel → tip knob → taper → nozzle → drop. Use viewBox="0 0 28 82" — the tall narrow canvas fits the upright dropper.
4
The drop is the action element. It sits below the nozzle, obeys gravity naturally (no rotation needed). Use a large teardrop — it must be prominent and visually separate from the barrel.
5
Glass highlight line. A single <line> running the height of the barrel in #F5F0E8 at low opacity gives the glass effect.
6
Test at multiple sizes. Always render at 46×108 and smaller. If it doesn't read clearly at half size, simplify.
The viewBox is 0 0 28 82 — narrow and tall, matching the upright dropper proportions. Center x=14.

App Architecture

Stack

  • Single self-contained HTML file — all CSS, JS, and seed data in one file
  • No framework, no build step, no npm — plain HTML/CSS/JS
  • Font: Plus Jakarta Sans via Google Fonts (200, 300, 400, 500 + italics)
  • Data: JSON stored in localStorage + Google Drive sync
  • Hosting: GitHub Pages

Data layer

  • Seed data hardcoded as JS constants in the HTML file
  • localStorage key: rituel_local — persists data between sessions
  • Data versioned with DATA_VERSION constant — bump when seed data changes to force reload
  • Google Drive file: rituel-data.json — auto-created on first sign-in
  • Auto-save debounced 1.2s after last change via scheduleSave()
  • Session token stored in sessionStorage to avoid re-auth on reload

Data structure

// Three top-level objects saved to Drive + localStorage:
{
  settings: {
    cycleStartDate: '',     // ISO date — Week 1 Monday anchor
    retinolNext: 'retinol'  // 'retinol' | 'granactive'
  },
  products: [ /* array of product objects */ ],
  log:      [ /* array of log entry objects */ ]
}

Key logic

  • Cycle: getCycleInfo(date) — takes a date, returns week number (1–4), isPeel, and evening routine type based on cycleStartDate
  • Retinol alternation: settings.retinolNext advances each time an evening Retinol session is logged
  • Depleted products: when Granactive is marked depleted, retinol step automatically uses Retinol 0.5% only; when Alpha Arbutin is depleted, step is struck through
  • Log: sessions logged via logSession(session, label); streak calculated from consecutive evening entries

Icon design process

  • Always provide a reference image before starting icon work
  • Claude draws upright first, rotates the group as a whole unit
  • Drop stays outside the rotation — gravity is always vertical
  • Share screenshots quickly — Claude cannot preview its own SVG output
  • Use simple primitives (rect, ellipse, circle) not freehand paths for the main body
  • Test at 48×60, 32×40, and 20×25 before finalising
